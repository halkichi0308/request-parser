Interfaces and classes used by the various implementations in the sub-packages.

Definitive implementations of commonly used encoders and decoders.

Codec is currently comprised of a modest set of utilities and a simple framework for String encoding and decoding in three categories: Binary Encoders, Language Encoders, and Network Encoders.

Binary Encoders
burp.com.org.apache.commons.codec.binary.Base64 Provides Base64 content-transfer-encoding as defined in RFC 2045 Production
burp.com.org.apache.commons.codec.binary.Hex Converts an array of bytes into an array of characters representing the hexadecimal values of each byte in order Production

Language Encoders

Codec contains a number of commonly used language and phonetic encoders

Language Encoders
burp.com.org.apache.commons.codec.language.Soundex Implementation of the Soundex algorithm. Production
burp.com.org.apache.commons.codec.language.Metaphone Implementation of the Metaphone algorithm. Production

Network Encoders

Codec contains network related encoders

Network Encoders
burp.com.org.apache.commons.codec.net.URLCodec Implements the 'www-form-urlencoded' encoding scheme. Production
